haroo-api

haroo-api for haroo cloud service

  • nodejs
  • restify
  • database bind
    • mongo : mongoose
    • couch : nano
    • mysql : sequelize

refer. package.json more.

Usage

should to ready mongo, couch and mysql database

install modules

npm install

npm test

npm start

Run App by MODE

NODE_ENV=development nodemon app

or

NODE_ENV=production pm2 start app.js --name "core"
